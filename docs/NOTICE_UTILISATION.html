Notice d'utilisation — Frixia

Guide pratique complet pour démarrer, configurer et exploiter Frixia efficacement.

1. Présentation de Frixia

Frixia est une plateforme multi-IA locale qui permet d’utiliser plusieurs modèles d’intelligence artificielle depuis une seule interface.

L’application fonctionne directement sur ton ordinateur à partir de fichiers web simples. Elle ne nécessite pas d’installation complexe, ni de base de données. Elle s’ouvre dans un navigateur internet à partir d’un fichier index.html.

Frixia permet notamment de :

  • dialoguer avec plusieurs IA depuis la même interface
  • utiliser des modèles texte, image et recherche web
  • changer de modèle en cours de conversation
  • enregistrer des prompts réutilisables
  • créer des rôles personnalisés
  • suivre précisément les coûts d’utilisation des API
  • sauvegarder et restaurer ses paramètres

2. Principe de fonctionnement

Frixia ne fournit pas directement les modèles IA. L’application sert d’interface entre toi et les fournisseurs d’IA comme OpenAI, Anthropic, Google ou Perplexity.

Pour utiliser Frixia, tu dois renseigner tes propres clés API. Ainsi, tu utilises directement les services des fournisseurs, sans intermédiaire.

Important : Frixia est gratuit en tant qu’outil, mais les appels API restent payants selon ta consommation. Tu ne paies donc pas un abonnement à Frixia, mais l’usage réel des modèles que tu appelles.

3. Installation de Frixia

Étape 1 : télécharger l’archive
Télécharge le fichier ZIP de Frixia.

Étape 2 : décompresser le fichier
Décompresse l’archive ZIP sur ton ordinateur dans le dossier de ton choix.

Étape 3 : ouvrir le dossier
Dans le dossier obtenu, tu trouveras plusieurs fichiers, dont un fichier principal nommé index.html.

Étape 4 : lancer l’application
Double-clique sur index.html pour ouvrir Frixia dans ton navigateur.

4. Stockage des données et sauvegarde

Frixia enregistre les paramètres directement dans le stockage local du navigateur.

Cela signifie que les réglages, prompts, rôles, préférences et clés API restent sur ton poste.

Attention : si tu effaces complètement les données du navigateur, tu peux perdre tes paramètres, tes prompts enregistrés, tes rôles et tes préférences.
Conseil : pense à utiliser régulièrement la fonction Exporter pour créer une sauvegarde de tes paramètres.

5. Import et export des sauvegardes

Frixia propose un système de sauvegarde simple.

  • Exporter permet de générer un fichier contenant les paramètres de l’application
  • Importer permet de recharger une sauvegarde précédemment exportée

Effectue une sauvegarde après avoir configuré tes clés API, créé plusieurs rôles, enregistré des prompts importants ou avant de changer de navigateur.

6. Configuration des clés API

Avant d’utiliser Frixia, tu dois renseigner au moins une clé API dans le menu Configuration.

Les fournisseurs prévus dans la présentation sont :

  • OpenAI
  • Anthropic
  • Google
  • Perplexity

Le principe est simple : tu crées une clé API depuis la plateforme du fournisseur, puis tu la copies dans Frixia.

Important : chaque clé est personnelle. Ne partage jamais tes clés API avec d’autres personnes.

7. Interface générale

L’interface de Frixia est organisée autour de plusieurs zones :

  • une colonne de gauche pour la configuration, les statistiques, l’import/export, les conversations, les dossiers, les prompts et les rôles
  • une zone centrale pour afficher la conversation en cours
  • une zone inférieure pour saisir un prompt, ajouter un fichier, activer la recherche web, utiliser la dictée vocale, améliorer un prompt et envoyer la demande

8. Créer une nouvelle conversation

Pour commencer, clique sur Nouvelle conversation.

Ensuite :

  • écris ton prompt
  • choisis le modèle à utiliser
  • active éventuellement la recherche web
  • clique sur envoyer

Exemple : Fais-moi un résumé de la chute du mur de Berlin

9. Choisir un modèle IA

Frixia permet de choisir différents types de modèles :

  • modèles texte pour les réponses, résumés, analyses, traductions et reformulations
  • modèles image pour générer des visuels à partir d’un prompt
  • modèles de recherche pour les recherches web enrichies

Tu peux changer de modèle même au cours d’une conversation.

10. Coût estimé et suivi des tokens

Frixia affiche plusieurs informations utiles :

  • le nombre de tokens en entrée
  • le nombre de tokens en sortie
  • le coût estimé
  • le temps de réponse

Cela permet de suivre précisément l’impact financier de chaque requête et de comparer les modèles.

11. Utiliser la recherche web

Active la recherche web lorsque ta demande nécessite des informations récentes ou vérifiables en ligne.

Évite de l’activer pour les sujets purement historiques ou généraux afin de ne pas générer de coûts inutiles.

12. Ajouter un fichier à une conversation

Frixia permet d’ajouter des fichiers à analyser, soit via le bouton d’ajout, soit par glisser-déposer.

Exemples d’usages :

  • analyser un document texte
  • exploiter un PDF
  • utiliser un contenu comme contexte pour une demande

13. Amélioration automatique du prompt

Une fonction d’optimisation permet d’améliorer automatiquement un prompt trop vague pour le rendre plus clair, plus précis et plus efficace.

Exemple de départ : Parle-moi du mur de Berlin

Exemple amélioré : Rédige un exposé clair et synthétique sur la chute du mur de Berlin en incluant le contexte historique, les causes, les événements et les conséquences

14. Dictée vocale et lecture audio

Frixia intègre des fonctions vocales :

  • dictée vocale pour parler au lieu d’écrire
  • lecture à haute voix pour écouter une réponse générée

Note : ces fonctions peuvent consommer des tokens et générer des coûts supplémentaires.

15. Gestion des conversations

Chaque conversation est conservée dans l’historique. Tu peux la rouvrir, la renommer, la supprimer ou la déplacer dans un dossier.

16. Organiser ses conversations avec des dossiers

Frixia permet de créer des dossiers personnalisés comme Marketing, Développement, Clients, Articles, Visuels ou Veille IA.

Tu peux ensuite glisser une conversation dans le dossier de ton choix.

17. Rechercher dans ses conversations

Une fonction de recherche permet de retrouver rapidement une ancienne conversation à partir d’un sujet, d’un mot clé ou d’une réflexion passée.

18. Enregistrer des prompts

Frixia permet d’enregistrer des prompts complets ou des morceaux de prompt pour les réutiliser plus tard.

Exemple : Réponds de manière claire, structurée et professionnelle

19. Insérer un prompt enregistré

Lors d’une nouvelle demande, tu peux insérer un prompt sauvegardé dans le champ de saisie afin de combiner une demande ponctuelle avec une consigne standardisée.

20. Les rôles

Les rôles sont l’une des fonctions les plus puissantes de Frixia. Un rôle agit comme une consigne générale appliquée à toute la conversation.

Exemples de rôles :

  • traducteur anglais
  • assistant marketing
  • professeur exigeant
  • rédacteur web
  • expert juridique
  • ami décontracté

21. Créer un rôle personnalisé

Tu peux créer tes propres rôles dans Frixia.

Exemple :

Nom : Rédacteur journalistique

Consigne : Tu rédiges avec clarté, précision et sobriété. Tu structures les réponses comme un article professionnel, sans blabla inutile.

Une fois enregistré, ce rôle peut être sélectionné au démarrage d’une nouvelle conversation.

22. Optimisation d’un rôle

Frixia permet d’optimiser automatiquement le texte définissant un rôle pour le rendre plus précis, plus structuré et plus efficace.

23. Exporter et importer des rôles

Les rôles peuvent être exportés dans un fichier puis réimportés plus tard.

  • sauvegarder tes rôles
  • les transférer vers un autre poste
  • les partager à d’autres utilisateurs

24. Exporter une conversation

Frixia permet d’exporter les conversations dans différents formats comme Markdown et HTML.

Le Markdown est pratique pour l’archivage et la réutilisation dans une autre IA. Le HTML conserve une mise en page plus propre pour la lecture et le partage.

25. Résumer automatiquement une conversation

Frixia peut produire un résumé synthétique d’une conversation complète pour garder l’essentiel, reprendre un sujet plus tard ou transmettre une synthèse à quelqu’un.

26. Génération d’images

Frixia permet d’utiliser des modèles d’image. Tu peux écrire un prompt visuel, choisir un format puis lancer la génération.

Formats mentionnés :

  • carré
  • paysage
  • portrait

27. Changer de modèle dans une conversation

Frixia permet d’utiliser plusieurs modèles dans une même conversation.

Exemple de workflow :

  • demander une analyse à un modèle texte avancé
  • demander un résumé à un modèle moins coûteux
  • générer un visuel à partir de la conversation
  • faire une recherche web complémentaire avec un modèle adapté

28. Statistiques d’usage

La section Statistiques permet de visualiser le coût total, les coûts par conversation, par modèle, par fournisseur et par catégorie.

Cela aide à piloter intelligemment ses dépenses API.

29. Fichier models.js

Un fichier nommé models.js permet de définir la liste des modèles disponibles dans Frixia.

Il peut être modifié manuellement pour ajouter ou supprimer des modèles et ajuster les informations de prix.

Attention : cette opération demande de respecter rigoureusement la structure du fichier pour éviter les erreurs.

30. Conseils de bonne utilisation

  • commence simple avec un seul fournisseur
  • sauvegarde régulièrement
  • surveille les coûts
  • n’active la recherche web que si nécessaire
  • crée tes rôles personnalisés
  • enregistre tes meilleurs prompts
  • teste plusieurs modèles selon les usages

31. Limites à connaître

  • dépendance aux API et aux clés valides
  • coût variable selon l’usage
  • risque de perte des données locales sans sauvegarde
  • présence possible de bugs selon l’évolution de l’outil

32. Conclusion

Frixia est une interface locale multi-IA simple, souple et puissante. Elle permet de centraliser plusieurs modèles dans un seul outil, tout en gardant la maîtrise des usages, des prompts, des rôles et des coûts.

Elle convient particulièrement aux utilisateurs qui veulent comparer plusieurs IA, payer à l’usage, personnaliser leur manière de travailler, organiser leurs conversations et générer du texte, des images et de la recherche depuis une seule interface.
