# Frixia

![Version](https://img.shields.io/badge/version-2.3-blue)
![Status](https://img.shields.io/badge/status-active-success)
![License](https://img.shields.io/badge/license-community-lightgrey)
![Platform](https://img.shields.io/badge/platform-web-local-orange)

**Frixia** est une plateforme **multi-IA locale** permettant d'utiliser plusieurs modèles d'intelligence artificielle dans une seule interface moderne.

L'application fonctionne directement sur votre ordinateur et utilise vos **clés API personnelles** pour accéder aux modèles d’IA comme :

- OpenAI  
- Anthropic  
- Google  
- Perplexity  
- Ollama

Aucun abonnement à la plateforme n’est nécessaire.

Vous payez uniquement **l’utilisation réelle des API**.

---

# Aperçu

Frixia est une interface web légère conçue pour centraliser l'utilisation de plusieurs intelligences artificielles dans un seul environnement.

Fonctionnalités principales :

- utilisation de plusieurs IA dans une seule interface  
- changement de modèle pendant une conversation  
- génération de texte et d’images  
- recherche web intégrée  
- prompts enregistrés  
- rôles personnalisés  
- suivi des coûts API  
- sauvegarde et restauration

# Fonctionnalités

## Conversations intelligentes

Chaque conversation peut utiliser un modèle différent.

Il est également possible de **changer de modèle pendant une conversation**.

---

## Prompts enregistrés

Frixia permet de créer une bibliothèque de prompts réutilisables.

Exemples :

- rédaction
- génération d’images
- marketing
- analyse de texte

Les prompts peuvent être insérés automatiquement dans une conversation.

---

## Rôles personnalisés

Les rôles définissent le comportement global de l’IA.

Exemples :

- traducteur
- expert marketing
- professeur
- assistant technique
- analyste

Un rôle agit comme un **prompt système permanent**.

---

## Génération d'images

Frixia permet de générer des images via les modèles compatibles.

Formats possibles :

- carré
- paysage
- portrait

Les images peuvent être téléchargées directement.

---

## Recherche web

Certains modèles permettent d'effectuer des recherches internet pour produire des réponses actualisées.

# Organisation des conversations

Les conversations peuvent être :

- renommées
- classées dans des dossiers
- recherchées
- supprimées

Cela permet d'organiser les projets efficacement.

---

# Sauvegarde

Frixia permet d’exporter l’ensemble des paramètres.

Le fichier de sauvegarde contient :

- configuration
- prompts
- rôles
- conversations

Vous pouvez ensuite restaurer ces données via l’option **importer**.

---

# Suivi des coûts

Chaque requête affiche :

- nombre de tokens
- temps de génération
- coût estimé

Les statistiques permettent également de suivre :

- coût par conversation
- coût par modèle
- coût par fournisseur

# Important

Frixia **n’est pas un fournisseur d’intelligence artificielle**.

L’outil utilise les **API officielles**.

Vous devez donc disposer :

- d’un compte chez les fournisseurs
- de crédits API disponibles
